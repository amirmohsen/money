    Currency

    Example:

    Using code
    let currency = new Currency('USD');
    Using existing currency object
    let currency = new Currency(new Currency('USD'));
    Using custom settings to create a one-off currency without registering it
    let currency = new Currency({
 code: 'ETH',
 symbol: 'Ξ'
});
