    public static ROUNDING: object source

    Rounding modes you can use in your operations. These map directly to BigNumber's rounding modes.

    Properties:

    NameTypeAttributeDescription
    ROUNDING.UP number

    Rounds away from zero
    ROUNDING.DOWN number

    Rounds towards zero
    ROUNDING.CEIL number

    Rounds towards Infinity
    ROUNDING.FLOOR number

    Rounds towards -Infinity
    ROUNDING.HALF_UP number

    Rounds towards nearest neighbour. If equidistant, rounds away from zero
    ROUNDING.HALF_DOWN number

    Rounds towards nearest neighbour. If equidistant, rounds towards zero
    ROUNDING.HALF_EVEN number

    Rounds towards nearest neighbour. If equidistant, rounds towards even neighbour
    ROUNDING.HALF_CEIL number

    Rounds towards nearest neighbour. If equidistant, rounds towards Infinity
    ROUNDING.HALF_FLOOR number

    Rounds towards nearest neighbour. If equidistant, rounds towards -Infinity

    Example:

    let
     price = new Money('7856', 'USD'), // $78.56
     discountedAndRoundedUp = price.multiply('0.70'), // $55.00
     discountedAndRoundedDown = price.multiply('0.70', Money.ROUNDING.DOWN); // $54.99

    public constructor(value: number | string | Money, currency: string | Currency) source

    Params:

    NameTypeAttributeDescription
    value number | string | Money

    integer, integer string, float string, instance of Money
    currency string | Currency

    currency code as string, instance of Currency

    Example:

    Integer as value
    let price = new Money(7856, 'USD'); // $78.56
    String integer as value
    let price = new Money('7856', 'USD'); // $78.56
    String Float as value
    let price = new Money('78.56', 'USD'); // $78.56
    Money instance as value
    let price = new Money(new Money('78.56'), 'USD'); // $78.56 - same as using clone()
    Currency instance as currency
    let price = new Money('7856', new Currency('USD')); // $78.56
