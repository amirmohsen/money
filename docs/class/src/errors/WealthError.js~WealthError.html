    WealthError

    Extends:

    es6-error~ExtendableError → WealthError

    Direct Subclass:

    CurrencyMismatchError, InvalidCurrencyError, WrongInputError

    The base error for all custom errors thrown by the "Wealth" library. It's useful when trying to catch all types of "Wealth" errors.

    Example:

    try {
 // operation
}
catch(e) {
 if(e instanceof WealthError) {
     // handle error
 }
}
